\subsection{Detailed Description}
wave climate objects for offshore wave conditions \begin{DoxyAuthor}{Author}
Martin D. Hurst, British Geological Survey
\end{DoxyAuthor}
This file contains objects for representing offshore wave conditions for the C\-O\-V\-E model. \hyperlink{classWave}{Wave} climates are described using wave direction (azimuth), significant wave height and wave period. \hyperlink{classWave}{Wave} climates can be intialised as a single wave conditions, a Gaussian distribution of direction, height and period, a bimodal distribution (i.\-e. two Gaussians) or using real wave climate data. \begin{DoxyDate}{Date}
26/10/2015 
\end{DoxyDate}
