RealWaveClimate::RealWaveClimate ( string  WaveFileName)
inline

Initialise function. Creates a real wave climate object from a specified text file.

Reads wave data from a text file. File format is three columns, the first is wave directions (azimuth), the second is wave periods (seconds) and the third is wave height (metres). The function assumes that the file as 1 header line.
