Detailed Description

wave climate objects for offshore wave conditions

Author
Martin D. Hurst, British Geological Survey

This file contains objects for representing offshore wave conditions for the COVE model. Wave climates are described using wave direction (azimuth), significant wave height and wave period. Wave climates can be intialised as a single wave conditions, a Gaussian distribution of direction, height and period, a bimodal distribution (i.e. two Gaussians) or using real wave climate data.
